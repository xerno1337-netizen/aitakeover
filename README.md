# AIScope - AI Job Takeover Radar

A modern, interactive dashboard tracking AI job displacement across the globe. Built with React and optimized for Vercel deployment.

## Features

- **Threat Radar**: Real-time visualization of job threat levels
- **Timeline Charts**: Projected automation rates from 2024 to 2040
- **World Map**: Country-by-country analysis of AI adoption
- **Job Analysis**: Detailed breakdown of automation timelines by profession

## Tech Stack

- React 18
- Vite
- Recharts (for data visualization)
- React Simple Maps (for world map)
- Modern CSS with clean white design

## Deployment to Vercel

1. Push your code to GitHub
2. Import the repository in Vercel
3. Vercel will automatically detect the Vite configuration
4. Deploy!

The `vercel.json` file is already configured for optimal deployment.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The built files will be in the `dist` directory, ready for deployment.
